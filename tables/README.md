# tables — DDL таблиц системы GP_ETL

Папка содержит DDL-скрипты для создания всех таблиц системы GP_ETL: инфраструктурных таблиц логирования, управляющих таблиц super workflow, конфигурационных таблиц ETL, таблиц Data Quality и Z-тестов, а также вспомогательных и системных таблиц. Каждый файл соответствует одной таблице и содержит полное определение структуры, включая партиционирование и индексы.

## Ключевые файлы

### Логирование ETL

| Файл | Описание |
|------|----------|
| `tb_log_workflow.sql` | Основная таблица журнала выполнения ETL-задач (старт, финиш, статус, строки) |
| `tb_log_workflow_err.sql` | Таблица ошибок ETL-задач с текстом исключений |
| `tb_log_workflow_stat.sql` | Агрегированная статистика по выполнению воркфлоу |
| `tb_log_ctl.sql` | Таблица контрольных точек ETL-процессов |
| `tb_log_ctl_all.sql` | Расширенная таблица контрольных точек с полной историей |
| `tb_log_work.sql` | Детальный лог рабочих шагов ETL (каждый DML-шаг) |
| `tb_log_skew.sql` | Лог данных о data skew таблиц |
| `tb_log_fld_stat.sql` | Статистика заполненности полей таблиц |
| `tb_log_pxf.sql` | Лог загрузок через PXF |

### Super Workflow

| Файл | Описание |
|------|----------|
| `tb_swf.sql` | Реестр всех задач super workflow с их метаданными |
| `tb_swf_status.sql` | Текущие статусы задач SWF |
| `tb_swf_hist.sql` | История изменения статусов задач SWF |
| `tb_swf__log.sql` | Основная таблица логов SWF-движка |
| `tb_swf_0_log.sql` — `tb_swf_9_log.sql` | Партиционированные логи SWF по группам (0–9) |
| `tb_swf_ctl_log.sql` | Контрольный лог событий SWF |
| `tb_swf_chk_log.sql` | Лог проверок качества в рамках SWF |
| `tb_swf_dia_log.sql` | Специализированный лог для DIA-загрузок в SWF |
| `tb_swf_pxf_log.sql` | Специализированный лог для PXF-загрузок в SWF |
| `tb_swf_mail_log.sql` | Лог отправки почтовых уведомлений |
| `tb_swf_log_wf.sql` | Лог воркфлоу-групп в SWF |

### Конфигурация и обмен данными

| Файл | Описание |
|------|----------|
| `tb_etl_config.sql` | Центральная конфигурационная таблица ETL-процессов |
| `tb_pxf.sql` | Конфигурация PXF-подключений к внешним источникам |
| `tb_exchange_log.sql` | Журнал операций обмена данными между системами |

### Data Quality

| Файл | Описание |
|------|----------|
| `tb_dq_checks.sql` | Реестр DQ-проверок с правилами и параметрами |
| `tb_dq_rule.sql` | Каталог правил Data Quality |
| `tb_dq_check_status.sql` | Текущие результаты DQ-проверок |
| `tb_dq_check_status_hist.sql` | История результатов DQ-проверок |
| `tb_dq_log.sql` | Детальный лог выполнения DQ-проверок |
| `tb_dq_stability.sql` | Метрики стабильности распределения данных |
| `tb_dq_nonnull_ratio.sql` | Метрики заполненности (доля непустых значений) |
| `tb_dq_max_date.sql` | Метрики актуальности данных (максимальная дата) |
| `tb_dq_aggregate_comp.sql` | Агрегированные DQ-метрики для сравнения |
| `tb_dq_user_sql.sql` | DQ-проверки на основе пользовательского SQL |
| `tb_dq_scd2_validity.sql` | Проверки корректности SCD2-историй |
| `tb_dq_date_not_in_range.sql` | Проверки выхода дат за допустимый диапазон |
