# dq — Data Quality фреймворк

Папка содержит фреймворк контроля качества данных (Data Quality) системы GP_ETL. Здесь реализованы функции расчёта DQ-метрик по различным методикам (стабильность, заполненность, актуальность, агрегаты, Z-тесты, пользовательский SQL), процедуры запуска проверок и логирования их результатов, а также инфраструктура статистического Z-тестирования. Фреймворк обеспечивает автоматический контроль качества данных на всех слоях хранилища.

## Ключевые файлы

### Функции расчёта метрик

| Файл | Описание |
|------|----------|
| `fn_dq_calc_stab_cont.sql` | Расчёт стабильности непрерывных (continuous) числовых показателей |
| `fn_dq_calc_stab_categ.sql` | Расчёт стабильности категориальных значений (распределение категорий) |
| `fn_dq_calc_stab_date.sql` | Расчёт стабильности временных рядов (распределение по датам) |
| `fn_dq_calc_nonnull_ratio.sql` | Расчёт доли непустых значений (Non-Null Ratio) для поля/таблицы |
| `fn_dq_calc_max_date.sql` | Расчёт метрики актуальности: максимальная дата в поле таблицы |
| `fn_dq_calc_aggregate.sql` | Расчёт агрегированных метрик (COUNT, SUM, AVG) для DQ-контроля |
| `fn_dq_calc_user_sql.sql` | Вычисление произвольной DQ-метрики через пользовательский SQL-запрос |
| `fn_dq_calc_not_in_range.sql` | Расчёт доли значений, выходящих за допустимый диапазон |
| `fn_dq_calc_ztest.sql` | Расчёт Z-статистики для сравнения текущего значения с историческим |
| `fn_dq_ztest_v2.sql` | Расчёт Z-статистики (версия 2, улучшенная обработка граничных случаев) |
| `fn_dq_calculate_interval_dt_from.sql` | Вычисление интервала дат для расчёта DQ-метрик |
| `fn_dq_delta_perc_prev.sql` | Расчёт процентного отклонения от предыдущего периода |

### Процедуры запуска проверок

| Файл | Описание |
|------|----------|
| `pr_dq_check.sql` | Основная процедура запуска DQ-проверок по правилам из tb_dq_checks |
| `pr_dq_nonnull_volume.sql` | Специализированная проверка заполненности и объёма таблицы |
| `pr_log_dq.sql` | Запись результатов DQ-проверки в таблицу логов |

### Z-тест инфраструктура

| Файл | Описание |
|------|----------|
| `pr_ztest_set.sql` | Настройка параметров Z-теста (эталонные значения, пороги) |
| `pr_ztest_recalc.sql` | Пересчёт Z-тестов за указанный период |
| `pr_ztest_all_diff.sql` | Расчёт Z-тестов для всех зарегистрированных метрик |
| `pr_ztest_add_error.sql` | Регистрация ошибки Z-теста в системе |
| `pr_ztest_add_except.sql` | Добавление исключения в Z-тест (игнорирование отклонения) |

### Вспомогательные функции

| Файл | Описание |
|------|----------|
| `first_day.sql` | Возвращает первый день месяца для заданной даты |
| `last_day.sql` | Возвращает последний день месяца для заданной даты |
| `add_month.sql` | Прибавляет N месяцев к дате (используется в расчётах интервалов) |
