# views — SQL-представления для мониторинга и аналитики

Папка содержит SQL-представления (VIEW), обеспечивающие удобный доступ к системной информации GP_ETL. Здесь сосредоточены представления для мониторинга логов ETL, отслеживания статусов super workflow, контроля обмена данными, анализа ресурсных групп Greenplum и мониторинга качества данных. Представления используются аналитиками и администраторами системы для оперативного контроля.

## Ключевые файлы

| Файл | Описание |
|------|----------|
| `vw_log_ctl.sql` | Сводное представление контрольных точек ETL — основной инструмент мониторинга загрузок |
| `vw_log_ctl_entity.sql` | Статусы загрузки в разрезе конкретных сущностей/таблиц |
| `vw_log_ctl_loading.sql` | Детализация процессов загрузки: время, строки, ошибки |
| `vw_log_ctl_wf.sql` | Логи в разрезе воркфлоу-групп |
| `vw_log_workflow.sql` | Общий журнал выполнения воркфлоу |
| `vw_log_workflow_err.sql` | Фильтрованное представление с ошибками воркфлоу |
| `vw_swf.sql` | Текущее состояние super workflow engine — главная панель мониторинга SWF |
| `vw_swf_status.sql` | Агрегированные статусы задач SWF (ожидание/в работе/готово/ошибка) |
| `vw_swf_ctl_log.sql` | Контрольный лог событий SWF-движка |
| `vw_swf_chk_log.sql` | Лог проверок качества в рамках SWF |
| `vw_swf__log.sql` | Детальный лог активности SWF |
| `vw_exchange.sql` | Представление для мониторинга обмена данными между системами |
| `vw_exchange_log.sql` | Журнал операций обмена данными |
| `vw_exchange_log_ids.sql` | Обмен данными в разрезе идентификаторов записей |
| `vw_exchange_log_keys.sql` | Обмен данными в разрезе ключей |
| `vw_resgroup_config.sql` | Конфигурация ресурсных групп Greenplum (CPU, memory) |
| `vw_resgroup_status.sql` | Текущее потребление ресурсных групп GP в реальном времени |
| `vw_dq_log.sql` | Журнал результатов проверок Data Quality |
| `vw_ztest.sql` | Результаты Z-тестов с историей и трендами |
| `vw_sberchat_dq.sql` | DQ-метрики для уведомлений в корпоративном чате |
| `vw_table_info.sql` | Метаданные таблиц: размер, количество строк, партиции |
