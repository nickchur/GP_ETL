# mail — Почтовые отчёты

Папка содержит процедуры формирования и отправки email-уведомлений и отчётов о состоянии ETL-системы. Здесь реализованы: базовая инфраструктура отправки почты, HTML-стилизация писем, конвертация SQL-результатов в HTML-таблицы и набор готовых тематических отчётов (контроль загрузки, маппинг, SDPUE, Z-тесты). Отчёты используются командой HR-аналитики для мониторинга и оперативного контроля качества данных.

## Ключевые файлы

| Файл | Описание |
|------|----------|
| `pr_send_mail.sql` | Базовая процедура отправки email через SMTP; принимает адресатов, тему и тело письма |
| `pr_mail_style.sql` | Функция генерации HTML-стилей для писем (CSS-шаблон корпоративного оформления) |
| `pr_tbl2html.sql` | Конвертация результата SQL-запроса в HTML-таблицу |
| `pr_tbl2html_loop.sql` | Итерационная конвертация нескольких результирующих наборов в HTML-таблицы |
| `pr_tbl2html_style.sql` | Конвертация в HTML-таблицу с применением стилевого оформления |
| `pr_mail_ctl_report.sql` | Отчёт о результатах прохождения контрольных точек ETL за период |
| `pr_mail_ctl_status.sql` | Письмо с текущим статусом ETL-задач (успех/ошибка/в процессе) |
| `pr_mail_ctl_work_load_report.sql` | Отчёт о нагрузке на систему загрузки данных |
| `pr_mail_mapping_check.sql` | Отчёт о результатах проверки маппинга полей источник→приёмник |
| `pr_mail_mapping_rebuild.sql` | Уведомление о пересборке маппинга между таблицами |
| `pr_mail_sdpue_report.sql` | Отчёт по данным SDPUE (кадровые события) |
| `pr_mail_informatica_report.sql` | Отчёт об интеграции с Informatica (статусы потоков) |
| `pr_mail_ztest_report.sql` | Отчёт по результатам Z-тестов (статистическое качество данных) |
