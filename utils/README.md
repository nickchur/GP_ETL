# utils — Утилитарные функции

Папка содержит вспомогательные SQL-функции общего назначения, используемые во всех остальных компонентах системы GP_ETL. Это базовый слой инфраструктуры: безопасное приведение типов, работа с JSON, получение служебных метаданных и утилиты записи в лог. Функции не содержат бизнес-логики и могут применяться в любом контексте.

## Ключевые файлы

| Файл | Описание |
|------|----------|
| `try_cast.sql` | Базовая функция безопасного приведения типов — возвращает NULL вместо ошибки при невалидном значении |
| `try_cast2int.sql` | Безопасное приведение к INTEGER |
| `try_cast2float.sql` | Безопасное приведение к FLOAT/NUMERIC |
| `try_cast2text.sql` | Безопасное приведение к TEXT |
| `try_cast2timestamp.sql` | Безопасное приведение к TIMESTAMP |
| `try_cast2bool.sql` | Безопасное приведение к BOOLEAN |
| `try_cast2json.sql` | Безопасное приведение к JSON |
| `try_cast2jsonb.sql` | Безопасное приведение к JSONB |
| `try_cast2uuid.sql` | Безопасное приведение к UUID |
| `try_cast2array.sql` | Безопасное приведение к ARRAY |
| `try_cast2regclass.sql` | Безопасное приведение к REGCLASS (проверка существования объекта БД) |
| `try_hash.sql` | Вычисление хеша от значения (используется для дедупликации и сравнения) |
| `clean_json_snapshot.sql` | Очистка и нормализация JSON-снапшота |
| `is_valid_json.sql` | Проверка валидности JSON-строки, возвращает boolean |
| `last_day.sql` | Возвращает последний день месяца для заданной даты |
| `set_search_path.sql` | Установка search_path для корректной адресации объектов схемы |
| `write_to_log.sql` | Низкоуровневая запись произвольного сообщения в таблицу логов |
| `pr_get_func.sql` | Получение метаданных о функции из системного каталога |
| `pr_get_part.sql` | Получение информации о партициях таблицы |
| `pr_get_last_pxf_key.sql` | Получение последнего ключа загрузки из PXF-источника |
| `pr_get_format_type.sql` | Определение формата типа данных колонки |
| `pr_table_size.sql` | Получение размера таблицы в читаемом формате |
| `readable_size.sql` | Форматирование байт в читаемый вид (KB/MB/GB) |
| `preprocess_text.sql` | Предобработка текстовых данных (нормализация, очистка) |
| `pr_cc_get_main_okved.sql` | Получение основного кода ОКВЭД для юридического лица |
| `pr_employee_hashtags_stat.sql` | Статистика по хэштегам сотрудников |
| `pr_zip_vasm_operation_cnt.sql` | Подсчёт операций по ВАСМ (вспомогательная утилита) |
