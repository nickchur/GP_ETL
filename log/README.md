# log — Логирование и контроль качества загрузки

Папка содержит инфраструктуру логирования ETL-процессов и процедуры контроля качества данных. Здесь сосредоточены все процедуры записи событий жизненного цикла ETL-задач (старт, завершение, ошибка, статистика), а также проверки корректности загрузки (уникальность ключей, счётчики строк, отсутствие NULL). Компоненты этой папки вызываются из всех ETL-пайплайнов и обеспечивают наблюдаемость всей системы.

## Ключевые файлы

| Файл | Описание |
|------|----------|
| `pr_log_start.sql` | Запись события старта ETL-задачи, возвращает идентификатор сессии логирования |
| `pr_log_end.sql` | Запись успешного завершения ETL-задачи с итоговой статистикой |
| `pr_log_error.sql` | Запись ошибки ETL-задачи с текстом исключения и стектрейсом |
| `pr_log_action.sql` | Запись произвольного действия/события в ходе выполнения задачи |
| `pr_log_work.sql` | Запись рабочей статистики (количество строк, время выполнения шага) |
| `pr_log_skew.sql` | Запись информации о data skew для таблицы в текущий момент |
| `pr_log_skew_hist.sql` | Запись исторических данных о skew таблиц |
| `pr_log_stat.sql` | Агрегация и запись итоговой статистики загрузки |
| `pr_log_last.sql` | Получение последней записи лога для задачи/таблицы |
| `pr_log_ctl.sql` | Контрольное логирование — запись контрольной точки ETL-процесса |
| `pr_log_event.sql` | Запись произвольного системного события |
| `pr_log_fld_stat.sql` | Статистика по отдельным полям таблицы (заполненность, уникальность) |
| `pr_log_pxf.sql` | Специализированное логирование для PXF-загрузок |
| `pr_log_informatica.sql` | Интеграция с логами Informatica |
| `pr_check_ctl.sql` | Проверка прохождения контрольных точек ETL-задачи |
| `pr_check_etl.sql` | Комплексная проверка корректности ETL-задачи |
| `pr_check_bd4ds.sql` | Проверка данных для слоя BD4DS |
| `pr_check_null.sql` | Проверка отсутствия NULL-значений в обязательных полях |
| `pr_check_mail_mapping.sql` | Проверка корректности маппинга для почтовых отчётов |
| `pr_chk_uniq.sql` | Проверка уникальности ключей в загруженной таблице |
| `pr_dia_chk_cnt.sql` | Сравнение счётчиков строк между слоями DIA и источником |
| `pr_scpl_json_update_is_parsed.sql` | Обновление флага разбора JSON в SCPL-процессах |
| `pr_scpl_json_update_schemas_settings.sql` | Обновление настроек схем SCPL |
| `pr_clean_ai_history.sql` | Очистка устаревших записей истории AI-обогащения |
| `pr_update_ai_history_i_pernr.sql` | Обновление истории AI-обогащения по табельному номеру (pernr) |
